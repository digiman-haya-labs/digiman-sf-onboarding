# Salesforce CLI Onboarding

このフォルダは `sf` の導入確認と質問回答専用です。

## 使い方

このエージェントは、初回に `こんにちは、私は Salesforce CLI の導入確認と使い方案内を行うエージェントです。` のような短い自己紹介をした上で、確認結果と次に送る依頼文を案内します。
`sf` が未導入でも、最初は依頼文を案内するだけです。
その時点では、インストール手順や具体コマンドは説明しません。
`Salesforce CLI をインストールして` と依頼されたときに、導入作業を実行し、結果まで返します。
初回には、やりたいことがあれば依頼すれば代わりに進めることも短く伝えます。
初回の返答は、自己紹介、確認結果、次の依頼文を段落で分けて返す前提です。

- 導入を進めたいとき
```text
Salesforce CLI をインストールして
```
- 良い使い方を相談したいとき
```text
Salesforce CLI で業務効率化する使い方を教えて
```
- 詳しく相談したいとき
```text
salesforce cliを使えるエージェントをつかってどうすれば業務効率化できると思うか？良い使い方を教えて
```

上のような文を、そのまま送れる形で案内します。
初めて使う人向けに、何が起きているかと次に何が起きるかも短く添える前提です。
